# Предсказание сердечно-сосудистых заболеваний: сравнительный анализ более 10 моделей и ансамблей

## Описание задачи
Соревнование Kaggle: https://www.kaggle.com/competitions/playground-series-s6e2/overview
Бинарная классификация для предсказания наличия сердечно-сосудистого заболевания у пациента. 
Метрика качества: ROC-AUC.

## Структура исследования

### 1: Разведочный анализ данных (EDA)
- Анализ пропусков и выбросов
- Визуализация распределений признаков
- Корреляционный анализ
- Выводы для дальнейшей работы

### 2: Предобработка
- Заполнение пропусков (обоснование метода)
- Масштабирование признаков
- Кодирование категориальных переменных

### 3: Базовое моделирование (до Feature Engineering)
- Обучены 6 базовых моделей:
  * Логистическая регрессия (baseline)
  * KNN
  * Random Forest
  * CatBoost
  * XGBoost
  * LightGBM
- Сравнение результатов
- Анализ корреляции предсказаний разных моделей

### 4: Ансамблирование (до FE)
- **Голосование (Voting Classifier)** — комбинация 3 бустингов
- **Стекинг (Stacking)** — мета-модель на основе бустингов
- **Блендинг (Blending)** — взвешенное усреднение
- Сравнение эффективности разных подходов к ансамблированию

### 5: Оптимизация гиперпараметров
- GridSearchCV для CatBoost (и других моделей)
- Влияние гиперпараметров на качество

### 6: Feature Engineering
**Созданные признаки:**
- [Признак 1] — [почему создан, какая логика]
- [Признак 2] — [почему создан, какая логика]
- [Признак 3] — [почему создан, какая логика]

**Визуализация новых признаков:**
- Графики распределений
- Анализ важности новых признаков

### 7: Моделирование после Feature Engineering
- Обучены 5 бустингов (добавились AdaBoost, GradientBoosting)
- Повторный перебор гиперпараметров
- Расширение пула моделей (SVM и др.)
- Анализ корреляции после FE

### 8: Финальные ансамбли
- Простое усреднение лучших моделей
- Сравнение результатов до и после FE
- Выбор финального решения
